Site Intent

The Angelfire.com re-design project will focus its efforts on creating a more usable and aesthetically pleasing user interface. We aim to clarify and make obvious the site's 3 main goals of offering users the following:

  1. A means to easily create websites
  2. Domain name & email addresses
  3. Website hosting services

Using clear entry points, we will be bringing Angelfire.com into the realm of modern web design. They will regain their lost marketshare in the template based website creation service industry.
